创业前先认清自己

发表于 2014-05-27   |   分类于 经验之谈   |     |   阅读次数

创业，但凡是互联网从业者都会想要去做的一件事，甚至不去做都不好意思说自己是做互联网的。这是每个人都逃离不了的怪圈。但是我们创业是为了什么？创业要去做什么？貌似我们都没一个很好的定义和觉悟。

在小的定义上，创业是在某个庞大的系统里钻空子，以小组织的形式快速而敏捷的撬走系统的某一肉饼。

在大的定义上，创业是一种创新，在众人皆睡我独醒的优势下，快速占据某一资源而获得生机。

通俗定义下，其实你就是想从苦逼的屌丝变成高大上的集权者而已（勿对号入座）。

入行四年，好多人都怂恿我去创业，甚至直接问我，你这么牛逼为什么不去创业，我都不知道怎么回答。我在创业公司呆过一年半，那段时光是快乐也很漫长的，快乐的是老板不断给你打鸡血，漫长是无止尽的加班。技术的广度是在这家公司训练出来，这里每个人都能独挡一面甚至多面。而且加班累积的时间也让我更快的变成了有经验的人。

人要有自知之明，至少在当下选你最适合做的事。你可以没事算算自己的机会成本，如果你有颗创业的心，那么你应该算同辈里的佼佼者，一旦你选择去创业，越优秀的你要放弃的东西就越多。也许你用创业的热情去打工，不出几年就变高管。也许你创业几年什么都没有，这个时候再重新回到大组织就各种不适应了。也许你创业几年，当上CEO，迎娶白富美，走上人生巅峰也说不定。当然这个创业的机会成本太高，不要随便浪费。

我自觉我不是一个天生的领袖，我只适合做一个帮领袖维护秩序的人。所以在我没创业之前，我努力做一个称职的管理者，等到有机会了我再去博一把，而且还有退路。而且我是一个过度聪明的人，做不了领袖。我只能配合那些有理想有天赋的人完成他们的心愿。不幸的是，至今我都还没成功帮人家完成心愿。而那些有理想但没执行力的人，就算你跟我说千万遍我也只能说最近好忙的，等过阵子再说。

好多人问我朋友找他去创业，问我应不应该去？在你去创业之前先问问自己，有没有钱，能不能维持生活，能不能不为五斗米折腰。你可能要说，“我TM去创业还不是因为没钱没权，不然我干嘛要去创业“。好吧，你既然连有钱有权的本事都没有，那你去了创业的小组织也不能担当重任，因为你想的只是钱权这么肤浅的东西，这个小组织有你没你也没什么区别呀（想想如果引进了风投，招了空降兵，哪里有那么多空闲的职位给你坐呢。）。你至少要在吃饱了穿暖了有点生活品味了才能说我TM还有更高更伟大的志愿吧！

你可能又会说，“哎呀，我只是作为一个技术入股，我只管做好产品就可以了，而且人家也给我开工资的，比我现在的还要高“。好吧，能这么想问题的你很蠢。首先，创业公司要不要加班？加班有没有加班费？你的薪水真的高？其次，你作为大系统下的一个小兵变成小组织下的大兵，变化有多大？相反的你的竞争也更大了，因为小组织里都是精英哦。别以为你在平凡人堆里很出众，到了小组织里一样与众不同哦。

好吧，这也不行那也不行，那到底什么人可以去创业？

如果你工作五年都没有，建议你提升事业的广度，找到最适合自己的位置，你可以去创业的小组织，因为那里能让你独挡多面，你也可以去大系统，聪明的你肯定想尽办法往上爬，如何让自己变得更加优秀，变成大家都需要的人。工作前五年你要赚的不是钱和权，把基础打好，对各种领域和行业有敏锐的嗅觉，肚子里要有货，同时要有一门精通的手艺在身。这样以后的日子才顺风顺水。

如果你工作五年＋，可以考虑考虑工作之外的机会，利用你庞大的优质人脉和信息渠道试水。作为领袖，一定要确保你迈出一大步，你身后有众多愿意帮助你的人支持你。作为维护领袖的管理者，一定要在领袖行动时跟上脚步，这样领袖成功了也意味着你成功。

社会之所以会进步，是因为有了明确的分工协作，如果这个社会每个人都去创业了，那社会就会混乱，每个人都有自己的位置，量力而行，做最好的自己做最对的自己才是你的使命。不管是大系统下或者小组织，无非都是人与人的连接，通过分工协作让自己让社会变得更加美好。别再纠结创业和大公司了。
